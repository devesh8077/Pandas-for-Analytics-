# Pandas-for-Analytics-
# 📊 Pandas & Feature Extraction using Python

## 📌 Overview

This repository contains my hands-on practice notebooks for learning **Pandas** and **Feature Extraction** using Python. These projects focus on real-world data manipulation, cleaning, and analysis techniques that are essential for Data Analytics.

## 🚀 Project Contents

### 1. Pandas Practice

In this notebook, I practiced:

* Creating Series and DataFrames
* Custom Labels and Indexing
* Data Selection and Slicing
* Adding and Removing Columns
* Handling Missing Values
* Filling Missing Data
* Merging and Joining DataFrames
* Basic Data Cleaning Operations

### 2. Feature Extraction

Using an Anime dataset, I performed feature engineering tasks such as:

* Creating new columns
* Extracting Episode Count
* Creating Timestamp Features
* Finding the Highest Scoring Anime
* Identifying Anime with the Most Episodes
* Top 5 Highest Rated Anime
* Top 5 Anime with the Highest Episode Count
